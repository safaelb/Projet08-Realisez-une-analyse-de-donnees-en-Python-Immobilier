## 🎯 Contexte
Accompagnement d'un client dans un changement de stratégie : analyser les tendances de prix de l'immobilier pour détecter des opportunités futures.

## 📚 Compétences développées
- Réaliser des analyses corrélatives et prédictives pour soutenir les décisions stratégiques
- Utiliser Python pour l'analyse de séries temporelles et la détection de tendances
- Présenter les résultats avec des visualisations claires et un discours argumenté

## 🧠 Ce que j'ai appris
Ce projet m'a permis d'aller plus loin avec Python : analyses de corrélations, tests statistiques et introduction aux modèles prédictifs. J'ai également travaillé la rigueur de la démarche analytique — de la problématique à la recommandation — en m'appuyant sur des données réelles du marché immobilier.

## 🗂️ Livrables
- Notebook Python (analyse exploratoire + modélisation)
- Présentation PowerPoint des résultats
- Recommandations stratégiques

## 🔧 Outils utilisés
python , excel , powerpoint
